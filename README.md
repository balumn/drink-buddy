# drink-buddy
An AI chatbot for drink suggestions

This bot can give you drink suggestions based on
 - your calender schedules (Google Calender + Outlook Calender)
 - based on the current weather
 - your taste + cocktail recipes if you want

### Future Improvements
 - Using Google Calender MCP + Outlook MCP: switch them to their public APIs for more handling
 - Location hard-coded to Kochi, Kerala: fetch it dynamically
